# ✅ FR-01 관리자 계정 / 권한 관리

### FR-01-01 관리자 계정 등록
- 관리자 계정을 생성할 수 있다.
- 비밀번호는 BCrypt로 암호화하여 저장한다.

### FR-01-02 관리자 로그인
- 로그인 ID + 비밀번호 인증 수행
- 계정 상태(status) 검증

### FR-01-03 관리자 상태 관리
- ACTIVE / INACTIVE / LOCK 상태 변경 가능

### FR-01-04 권한 그룹 생성
- 권한 그룹 등록 (group_key UNIQUE)

### FR-01-05 권한 그룹 수정/삭제
- Soft delete 기반 삭제
